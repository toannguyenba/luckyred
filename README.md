# Lucky Red - Mở lì xì may mắn

Một ứng dụng web tĩnh nhỏ để quay lì xì may mắn.

Files:
- `index.html` - UI và canvas bánh quay
- `styles.css` - Giao diện
- `app.js` - Logic bánh quay, xử lý input, animation, confetti

How to run:
1. Mở `index.html` trong trình duyệt (double-click) hoặc chạy một static server.
   - Windows PowerShell quick server (Python required):

```powershell
python -m http.server 8000
```

Open http://localhost:8000

Edit the amounts in the input (comma separated) or use presets, then click "Quay".
